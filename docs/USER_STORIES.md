# User Stories

## Epic: Administrative Management

### US01 - User Management
**As an** administrator,
**I want to** register new users and define their profiles,
**So that** they can access the system with appropriate permissions.

### US02 - Managerial Dashboard
**As a** manager,
**I want to** view a dashboard with key indicators,
**So that** I can track company performance in real-time.

## Epic: Commercial Operation

### US03 - Client Registration
**As a** commercial operator,
**I want to** register new clients with their complete data,
**So that** I can keep the contact base updated.

### US04 - Negotiation Recording
**As a** commercial operator,
**I want to** record the progress of a negotiation,
**So that** I can keep the history and follow-up organized.

## Epic: Production

### US05 - Service Order
**As a** production operator,
**I want to** view pending service orders,
**So that** I know which tasks to prioritize.
