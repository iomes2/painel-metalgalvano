# Functional Requirements

## Authentication and Users
- **FR01**: The system must allow users to log in with email and password.
- **FR02**: The system must differentiate user profiles (Admin, Manager, Operator).
- **FR03**: The system must allow user registration and editing (Admin only).

## Client and Negotiation Management
- **FR04**: The system must allow registration, editing, and deletion of clients.
- **FR05**: The system must allow recording negotiations linked to clients.
- **FR06**: The system must allow viewing the negotiation history.

## Service Orders
- **FR07**: The system must allow the creation of service orders.
- **FR08**: The system must allow updating the status of service orders.
- **FR09**: The system must generate service order reports.

## Reports
- **FR10**: The system must provide performance and sales reports.
- **FR11**: The system must allow exporting reports to PDF.
